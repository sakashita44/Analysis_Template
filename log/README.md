# log

## 留意事項

### レベルの種類

以下の情報レベルを付与したうえで，ログを出力することが望ましい

* `INFO`: 情報
    * 次の解析に移行する等，解析の進捗状況等を示す場合
* `WARN`: 警告
    * 解析対象ファイルが存在しない場合等，次の処理に進むことはできるが，正常な処理ができない場合
* `ERROR`: エラー
    * 解析に必要なデータが存在しない等，解析が続行できない場合
